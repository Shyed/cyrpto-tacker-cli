# Crypto Tracker (CLI)

A simple command-line app to fetch real-time cryptocurrency prices (BTC, ETH) using the CoinGecko API.

## Features

- Fetches live price in USD
- Supports Bitcoin, Ethereum (others with minor edits)
- No API key required

## API Used

- [CoinGecko Public API](https://www.coingecko.com/en/api)

## Sample Output
=== Crypto Tracker ===
Available coins: bitcoin, ethereum
Enter the cryptocurrency name (e.g. bitcoin): ethereum
Fetching price...

Current ETHEREUM price: $2581.01 USD
